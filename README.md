# Wedding-Plannerr



# Tech Stack used
- React
- Tailwind
- Framer motion
- Nodejs
- Express
- Mongodb
- Vercel (deployment of Frontend) + Fly (deployment of Backend)

### Prerequisites
- Node.js (v18 or higher)
- bun

### Live URL
You can access the live application at: [https://wedding-plannerr.vercel.app/](https://wedding-plannerr.vercel.app/)

### Installation

#### Frontend
1. Navigate to the frontend directory:
    ```sh
    cd wedding-frontend
    ```
2. Install dependencies:
    ```sh
    npm install
    ```

#### Backend
1. Navigate to the backend directory:
    ```sh
    cd wedding-backend
    ```
2. Install dependencies:
    ```sh
    bun install
    ```

### Running the Project

#### Frontend
1. Start the development server:
    ```sh
    npm run dev
    ```
2. Open your browser and navigate to `http://localhost:3000`

#### Backend
1. Start the development server:
    ```sh
    bun run index.ts
    ```

### Building for Production

#### Frontend
1. Build the project:
    ```sh
    npm run build
    ```
2. The production-ready files will be in the `dist` directory.

### Running Tests

#### Frontend
1. Run the tests:
    ```sh
    npm test
    ```

### Linting

#### Frontend
1. Lint the code:
    ```sh
    npm run lint
    ```

### Formatting

#### Frontend
1. Format the code:
    ```sh
    npm run format
    ```
